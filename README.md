# Employee Salary Excel Analysis

## Project Overview
This project demonstrates how essential Excel formulas and functions can be used to analyze an employee salary dataset and answer common business questions.

Using Microsoft Excel, several analytical tasks were performed such as identifying high-salary employees, analyzing departmental salary distribution, retrieving employee information, and generating summary insights.

The goal of this project is to showcase practical Excel skills used in real-world data analysis.

---

## Dataset Description
The dataset contains employee information including:

- Employee ID  
- First Name  
- Last Name  
- Gender  
- Department  
- Salary  
- Salary Bucket  
- Start Date  
- FTE (Full-Time Equivalent)  
- Employee Type  
- Work Location  
- Tenure  
- Work Type  

---

## Business Questions Solved

1. Total salary and headcount by department  
2. All employees with more than \$100k salary  
3. Female employees with more than \$100k salary  
4. Lowest, highest and top 5 salary values  
5. List of all departments  
6. Employee details lookup  
7. Employee lookup with error handling  
8. Identify the employee with the highest salary  
9. Employees who joined in a specific month (March)  
10. Department-level report including headcount, salaries, and percentage difference from overall average  

---

## Excel Functions & Techniques Used

- SUMIFS  
- COUNTIFS  
- FILTER  
- MIN / MAX  
- LARGE  
- SORT  
- TAKE  
- UNIQUE  
- COUNTA  
- VLOOKUP  
- INDEX + MATCH  
- XLOOKUP  
- IFERROR  
- MONTH  
- Conditional Formatting  

These functions were used to transform raw employee data into meaningful insights.

---

## Key Analysis Examples

- Identifying employees earning above \$100,000  
- Calculating departmental salary totals  
- Retrieving employee information using lookup functions  
- Finding the highest-paid employee  
- Filtering employees based on salary and gender  
- Creating department-level summaries using aggregation functions
